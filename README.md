# 🍔 BurguerFlow - Sistema de Gestão de Hamburgueria

Este projeto é um protótipo funcional desenvolvido para a disciplina de **Modelagem de Sistemas (DCC117)** na UFJF.

## 📋 Sobre o Projeto
O BurguerFlow visa otimizar o atendimento interno de hamburguerias, integrando o lançamento de pedidos (Garçom), a fila de produção (Cozinha) e o fechamento de contas (Caixa).

## 🛠️ Tecnologias
- **HTML5 / CSS3**: Estrutura e estilização.
- **Bootstrap 5**: Layout responsivo e componentes.
- **JavaScript**: Lógica de navegação e prototipagem funcional.

## 📁 Estrutura de Arquivos
- `index.html`: Menu principal de seleção de perfil.
- `garcom.html`: Gestão de mesas e pedidos.
- `cozinha.html`: Painel de monitoramento da produção.
- `caixa.html`: Interface financeira para fechamento de mesas.
- `estilo.css`: Padronização visual do sistema.
